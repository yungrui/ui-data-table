data-table.js Component Showcase

A lightweight, zero-dependency, feature-rich and highly customizable pure JavaScript Web Component for data tables. It comes with built-in Bootstrap 5-like default styles, ensuring a professional appearance in any project.

API Overview

HTML Attributes (Set via <ui-data-table> tag)

These attributes provide the most direct way to configure table behavior statically in HTML.

AttributeDescriptionDefaultExample
data-source-type Specifies the type of data source.
  • local: Uses a local array passed via setData(). Sorting, pagination, and filtering are handled on the client side.
  • server: Enables full server-side processing. All operations send requests to fetch-data-url.

server-total-fieldField name for the total record count in the API response.totalCountserver-total-field="total"

sort-orderThe initial sort order (asc or desc).ascsort-order="desc"

JavaScript Properties & Methods
Prop/MethodDescriptionExample
columns (Required) Array of column definitions.
  • field (string): (Required) Data key.
  • title (string): Header text.
  • sortable (boolean): Enable sorting.
  • render (function): Custom cell renderer (returns HTML string or DOM).
  • isCheckbox (boolean): Renders as a checkbox for selection.
table.columns = [
  { field: 'id', title: 'ID', sortable: true },
  { field: 'name', title: 'Name' }
];
setData(...) (Async) Primary method to load data.
  • keepHeaders (boolean): (New) Merge new headers with existing ones.
await table.setData(myArray);
updateHeaders(headers) (New) Merges new HTTP headers (e.g., Auth token) without a full re-render. 
table.updateHeaders({ 'Auth': 'Token' });
Custom Events (New) Standard events: row-click, selection-change, page-change, sort-change. 
table.addEventListener('row-click', (e) => {
  console.log(e.detail.rowData);
});
CSS Shadow Parts

Available parts: pager-container-top, pager-container-bottom, table-container, table, table-header, table-body, table-header-cell, table-cell, loading-overlay, loading-spinner.
